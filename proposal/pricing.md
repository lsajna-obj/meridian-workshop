# Pricing

**Proposal in response to RFP MC-2026-0417**

---

## Fee Structure

We are proposing a **fixed fee** for the required scope (R1–R4). Fixed fee gives Meridian cost certainty; it reflects our confidence in the estimate, which is based on review of the actual codebase rather than the RFP alone.

---

## Required Scope (R1–R4)

| Item | Description | Fee |
|------|-------------|-----|
| R3 | Automated browser test suite — all main views | $8,000 |
| R1 | Reports module audit and full remediation | $12,000 |
| R2 | Restocking recommendations — backend endpoint + frontend view | $18,000 |
| R4 | Architecture documentation | $4,000 |
| **Total — Required** | | **$42,000** |

---

## Desired Scope (D1–D3)

Desired items are priced separately and can be added to the engagement at contract execution or at the Phase 3 kickoff.

| Item | Description | Fee |
|------|-------------|-----|
| D2 | i18n extension to remaining modules | $5,000 |
| D1 | UI modernization (design alignment, component refresh) | $7,000 |
| D3 | Dark mode (operator-selectable theme) | $4,000 |
| **Total — Desired (all three)** | | **$16,000** |

---

## Not-to-Exceed

| Scenario | Total |
|----------|-------|
| Required scope only (R1–R4) | $42,000 |
| Required + all desired (R1–R4, D1–D3) | $58,000 |

Both figures fall within the $40,000–$60,000 range confirmed by Meridian.

---

## Terms

- **Payment schedule:** 30% at contract execution, 40% at Phase 2 exit (R1 and R2 accepted), 30% at final handoff.
- **Out-of-scope work:** Any work outside the defined scope will be agreed in writing before it begins, at a rate of $175/hour.
- **Defects found during R1 audit:** Remediation of all defects discovered during the Reports audit is included in the R1 fixed fee, regardless of the final defect count.

---

## Assumptions Affecting Price

- Flat JSON data files remain the data layer; no database migration is in scope.
- Meridian provides one designated point of contact available for weekly reviews.
- The codebase provided with the RFP package is the current production state.
- D1 design direction is confirmed at kickoff; rework driven by scope change after alignment is billable.
